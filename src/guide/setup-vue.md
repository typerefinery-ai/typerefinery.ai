## Overview

This template should help get you started developing `Electron` app using `Vue 3`, `Vite`, `Vuex` and `primeVue`.

## Run Setup

```sh
# clone the project
git clone https://github.com/typerefinery-ai/typerefinery.ai.git

# enter the project directory
cd typerefinery

# install dependency
npm install

# develop
npm run electron:dev
```
