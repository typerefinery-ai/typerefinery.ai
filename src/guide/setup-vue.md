## Run Setup

```sh
# clone the project
git clone https://github.com/typerefinery-ai/typerefinery.ai.git

# enter the project directory
cd typerefinery

# install dependency
npm install

# develop
npm run electron:dev

# build the app and run
npm run electron:start
```
