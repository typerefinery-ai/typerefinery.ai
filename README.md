## Run Setup

```sh
# clone the project
git clone  https://github.com/typerefinery-ai/typerefinery.ai.git

# install dependency
npm install

# develop
npm run dev
```

## Run Crawl

All command are tested in powrshell 7.

1. Start dev server

```powershell
npm run dev
```

2. Run crawl

```powershell
./algolia-crawl.ps1 
```
